UAV SEARCHING WITH DRONE NETWORKS

Dissertation Project

Unity, C#

Github Repo

Description

My dissertation looked at the effectiveness of drones for search and rescue operations. A previous dissertation had already provided a simulation, however it had no visualisation. I decided I would create a 3D visualisation in Unity of the flight data from the simulation and, on top of this, generate an orthomosaic map of a given area using images from the onboard cameras of the drones. This orthomosaic map would be used to bolster the evidence that drones are a viable option for search and rescue missions.

THEY CAME FROM THE WOODS

Game

Unity, C#

Github Repo Play it here

Description

This is a game I created for a module I took in my 3rd year at university. It is a top down, retro style, twin-strick survival shooter set in the woods. A blood moon has risen and the trees have come to life, you must survive the hordes and make it to the end of 5 levels to win. Along the way you can pick up coins used to purchase upgrades. There is also an endless mode. It was made completely in Unity using assets from opengameart.org.

TROPICAL ISLAND GRAPHICS

University Graphics Coursework

C++, OpenGL, GLSL

Github Repo View it here

Description

As part of my Masters in Games Engineering I participated in a graphics module, the end product of which you can view on my github or on youtube. The scene I created uses a multitude of modern graphical techniques. The terrain generated used multiple noisemaps for a realistic rough ground, multiple textures were used depending on height and direction. I created realistic water using bumpmapping and envrionment mapping. Multiple light sources were used in the scene using shaders, bloom was added in post-processing to give the lights glow. Multiple shadows are available through the use of a single, giant shadow map that holds multiple smaller shadow maps. A minimap was also created using multiple render textures. A more thorough list of features is available on my youtube channel, simply click the 'view it here' button above.

C++ PHYSICS ENGINE

University Coursework

C++

Github Repo View it here

Description

As part of my Masters in Games Engineering I participated in a physics module. The repo and a youtube video showing off the aspects of my coursework are linked above. The coursework consisted of creating a replica of "Screwball Scramble" by creating your own physics engine. To achieve this I implemented collision detection for Capsules, Spheres, Rays, Planes, and Cubes - both Axis Aligned and Oriented, as well as Collision Resolution for each combination. I also implemented multiple types of collision resolution, projection/impulse and a penalty method using Hooke's Law. The second part of the coursework involved implementing AI. I created two types of AI, one to be the enemy in a maze level using Heirarchal State Based Artificial Intelligence, and a simple Pushdown Automata to control the Menus.
